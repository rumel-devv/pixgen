# 🚀 PixGen — AI Photo Generator Platform

PixGen is a modern AI-powered photo generation web app built with **Next.js, Tailwind CSS, and Better Auth**.
Users can generate, explore, and download AI-generated images with a clean SaaS-style UI.

---

## ✨ Features

* 🔐 Authentication (Email + Google Login)
* 🖼️ AI Photo Gallery
* 📄 Photo Details Page
* ❤️ Like & ⬇️ Download system (UI ready)
* 🔍 Category-based browsing
* 🧑 User Profile page
* 💳 Pricing Page (static UI)
* 🔒 Protected routes (middleware/proxy)

---

## 🛠️ Tech Stack

* **Frontend:** Next.js (App Router)
* **Styling:** Tailwind CSS
* **UI Library:** HeroUI
* **Auth:** Better Auth
* **Icons:** React Icons
* **Deployment:** Vercel
